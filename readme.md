# Project Overview: Cinematic Room

**Project Title:** Cinematic Room

**Subtitle:** Shooter: The Sniper's Cabin

**Objective:** To develop a website where users can explore and interact with the 3D-modeled environment of the cabin from the movie "SHOOTER," enhancing the immersive experience of the film.

**Key Features:**
- **3D Modeling:** Realistic 3D recreation of the cabin where the protagonist of "SHOOTER" resides, based on the actual set from the movie.
- **Interactive Elements:** Features allowing users to interact with various objects within the cabin.
- **Information Provision:** Offers information and behind-the-scenes stories related to the objects in the cabin.
- **Support for Multiple Movies:** Provides different rooms for various movie titles and continuously updates with new content.
